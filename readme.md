# linux cheat sheet 


# pwd 
tells you current directory 

# ls 
lists files in your current directory

# cd 
changes your directory

# mkdir 
makes new director

# rmdir 
removes director

# touch 
makes an empty file

# rm 
removes a file

# cp
to copy a file

# mv 
moves files or rename

# cat 
show content of the file 

# less
shows file content one page at a time
